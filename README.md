## Project Overview

I Build a Professional Teleprompter App with React Native – iOS & Android (Full Build)

In this video, I build a full-featured teleprompter app from scratch using
React Native and Expo — inspired by Detail.co. The app lets you write or
paste a script, auto-scroll it while recording with your camera, and export
the video directly to your camera roll.

## 🔥 Features covered:

- • Script writing & library (save, edit, delete, import from clipboard)
- • Auto-scrolling teleprompter with adjustable speed
- • Floating & draggable teleprompter overlay
- • Word highlight (karaoke-style) while recording
- • Mirror mode for teleprompter glass setups
- • Live speed control during recording
- • Video quality selector (720p / 1080p / 4K)
- • Countdown timer before recording starts
- • Save & share recorded video
- • Beautiful dark UI with purple accent theme

## 🛠 Tech Stack:

- • React Native + Expo SDK 54
- • expo-camera for video recording
- • expo-media-library for saving videos
- • expo-video for playback
- • React Navigation (Stack)
- • AsyncStorage for script persistence
- • Expo Haptics, Keep Awake, Clipboard
